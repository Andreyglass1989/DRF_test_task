Т.З. 
Надо создать CRUD(create/read/update/delete) приложение на базе Django Rest Framework.
 где будет возможность управлять "Товар", то-есть, 
 создавать/редактировать/визуализировать/удалять его.


Товары должны иметь следуюшие данные:
1. Название
2. Цена
3. Изображение (одно или больще)
4. Автор (кто создал товар)
5. Дата создания

Для приложения база данных должна быть Postgresql
